# 每日待办管理器

一个为个人效率打造的 Web 版 To-do List，支持任务录入、筛选、统计以及导入导出。数据完全存储在浏览器本地，方便安全地管理每天的计划。

## 功能亮点

- 🗂️ **多视角列表**：按今天、将来、逾期、已完成或全部任务快速切换视图。
- 🧠 **结构化记录**：支持填写标题、详细说明、截止日期、优先级与标签，帮助你更好地规划每一个任务。
- 📊 **仪表盘统计**：实时显示今日任务、待处理总量、已完成数量以及逾期提醒。
- 🔄 **高效操作**：勾选即完成，支持一键延期、编辑与删除，同时提供批量清除已完成任务的能力。
- 💾 **数据迁移**：提供导入/导出 JSON 的功能，随时备份或迁移到其他设备。

## 使用方式

1. 克隆或下载本仓库代码：

   ```bash
   git clone <your-repo-url>
   cd test1
   ```

2. 使用任意静态服务器或直接通过浏览器打开 `index.html` 文件即可开始使用。例如使用 VS Code Live Server 或者：

   ```bash
   npx serve .
   ```

3. 首次打开会自动生成示例任务，熟悉操作后可以删除或覆盖。

## 技术实现

- 纯前端实现，使用 HTML + CSS + 原生 JavaScript，无需后端即可运行。
- 任务数据保存在 `localStorage` 中，刷新页面也不会丢失。
- 响应式布局，适配桌面端与移动端浏览体验。

## 数据备份与迁移

- 点击右上角的「导出数据」按钮，会下载包含所有任务数据的 JSON 文件。
- 点击「导入数据」可以选择之前导出的 JSON 文件，导入后会覆盖当前任务列表。

> 本项目主要面向个人日常任务管理，可在此基础上继续扩展，例如加入提醒、同步或团队协作等功能。
